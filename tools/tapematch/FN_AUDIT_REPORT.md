### Methodology notes

- FN population = frozen-set positives (`regression_set.json`) the current committed `config.yaml` still verdicts `different_family`, computed via the same `tapematch.verdict.cluster_verdicts` path `regression.py score --cached` uses, restricted to `corr < 0.05`.
- `label_assessment` heuristic (transparent, not a black box — reviewers should spot-check): `suspect-label` if source_texts name disjoint taper handles, or the speed-corrected performance-duration ratio is >15% off unity; `plausible-same-lineage` if format differs (one side MP3), or hf_ceiling gap >1 kHz with the narrow side <8 kHz (band-limited generation), or the curator relation text affirmatively links the two LBs, or the quick envelope check mean corr >=0.30; else `indeterminate`.
- The 4-band envelope quick check is a throwaway heuristic (single mid-track window, +/-20s lag search, best-available speed ratio only when both sides carry a reliable `speed_kind`) — supporting evidence only, never decisive alone for `suspect-label`.
