What it does

Everything a collector needs.

All the tools you need to work with the LosslessBob archive — in one native desktop app.

🔍

Checksum Lookup

Paste or load FFP, MD5, or ST5/SHA1 checksum files and instantly identify which LB entry a recording belongs to. See match quality, cross-references, and missing or duplicate files at a glance.

🗺️

Interactive World Map

Explore the entire archive on an interactive Leaflet map. Every recording plotted by concert location — zoom in, cluster, click for details. Heatmap view shows where Bob played most.

🔎

Full-Text Search

Search across all entry metadata — dates, venues, setlists, descriptions. Results update live as you type, with sortable columns and one-click detail view.

✅

File Verification

Point the app at your local audio files and verify them against the official LosslessBob checksums. Broken, missing, or extra files are highlighted immediately.

📂

Folder Renamer

After a lookup, propose standard LosslessBob-compliant folder names for your recordings. Preview renames before committing, with a full rename history log.

📡

Offline Archive Mirror

Spider and cache the entire losslessbob.com site locally. Browse entry pages, attachments (.ffp, .txt, .md5), and the LBBCD bootleg catalog — no internet required after the initial crawl.

🎵

Spectrogram Viewer

Generate SoX spectrograms for any audio file directly from the app. Spot upsamples, lossy re-encodes, or anomalies without leaving LosslessBob Desktop.

💿

Bootleg-CD Catalog

Browse the full LBBCD bootleg CD index — search by entry, date, or label. Scraped directly from losslessbob.com and stored locally for offline use.

🌐

Torrent Generation & qBittorrent

Create .torrent files for any LB entry with one click. Directly add them to qBittorrent via its WebUI API — no manual drag-and-drop required.

📱

Mobile-Friendly Admin Panel

Access a full-featured admin control panel from any device on your local network — trigger imports, check status, manage the database from your phone.

🎨

Color Themes

Choose from built-in themes or build your own color scheme with the live theme editor. All colors persist across sessions.

🗄️

Database Editor

Power-user table browser: view, edit, delete any row, and export to CSV. Automatic backup snapshots keep your data safe.

Get started

Install in minutes.

Python 3.11+ required. Install dependencies, run once, and the GUI opens automatically.

🪟 Windows

git clone https://github.com/kuddukan42/losslessbob
cd losslessbob
pip install -r requirements.txt
python main.py

A standalone .exe build is planned — see the GitHub releases page.

🍎 macOS

git clone https://github.com/kuddukan42/losslessbob
cd losslessbob
pip3 install -r requirements.txt
python3 main.py

Tested on macOS 13+. PyQt6 wheels available on pip — no Homebrew Qt needed.

🐧 Linux

git clone https://github.com/kuddukan42/losslessbob
cd losslessbob
pip install -r requirements.txt
python main.py

Debian/Ubuntu users may need apt install libxcb-cursor0 for Qt to start.

Optional: Install SoX for spectrogram generation, and configure qBittorrent credentials in the Setup tab for one-click torrent adding.
