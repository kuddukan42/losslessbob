The LosslessBob archive,

A modern desktop app for collectors of Bob Dylan lossless recordings — checksum lookup, a browsable library of every known show, collection tracking, verification, maps, and more. Electron/React frontend over a local Flask backend; your data never leaves your machine.

What it does

Everything a collector needs.

All the tools you need to work with the LosslessBob archive — in one native desktop app.

🔍

Checksum Lookup

Paste or drop FFP, MD5, or ST5/SHA1 checksum files and instantly identify which LB entry a recording belongs to — 700,000+ checksums indexed locally. Match status and cross-references at a glance.

📚

Library Browser

Every known Dylan show in one browsable timeline — flip between performance and recording views, filter by decade, coverage, or rating, and open a full dossier for any show: recordings, tapers, setlists, picks.

✅

Collection Pipeline

Point the app at your folders: it verifies audio against official checksums, identifies each recording, proposes standard LB folder names, and tracks what you own, want, and are missing.

🔎

Faceted Search

Search across all entry metadata — dates, venues, setlists, descriptions — with facet filters for decade, status, rating, type, and ownership. Save any filter combo as a one-click view.

🗺️

Interactive World Map

Every concert plotted by geocoded venue on an interactive Leaflet map — cluster markers, decade colors, heatmap overlay, and per-venue show lists linked back to search.

📅

Gaps View

A living uncirculated-shows list in the tradition of Les Kokay: every known concert date colored by whether a recording circulates, so you can see exactly what's still missing from the archive.

🎚️

TapeMatch Families

Acoustic fingerprint matching groups recordings of the same show into source families — see which LB entries share a tape lineage and which are genuinely distinct sources.

🎵

Spectrogram Viewer

Batch-generate SoX spectrograms for any folder of audio directly from the app. Spot upsamples, lossy re-encodes, EQ tells, and dropouts without leaving LosslessBob Desktop.

💿

Bootleg-CD Catalog

Browse the full LBBCD bootleg CD index — 1,380 titles searchable by entry, date, or label, stored locally for offline use.

🌐

Torrents & qBittorrent

Create .torrent files for any LB entry with one click, or add them straight to qBittorrent via its WebUI API — no manual drag-and-drop.

🤝

Trading & Sharing

Build trade lists from your collection, compare against a trading partner's list, and share filtered views of what you have and what you're hunting for.

🎨

Themes & Languages

Built-in color themes plus a live theme editor, and a fully translated interface — English, German, French, Spanish, Italian, and Dutch.

Get started

Install in minutes.

Download an installer, launch, and follow the first-run wizard — it downloads the full dataset and points you at adding your collection folders. No manual imports needed.

🪟 Windows

LosslessBob-<version>-windows-Setup.exe

Grab the installer from the latest release — a portable .exe (no install) is also available.

🐧 Linux

LosslessBob-<version>-linux-x86_64.AppImage
chmod +x LosslessBob-*.AppImage

Single-file AppImage — make it executable and run. No packages to install.

🛠️ From source (any OS)

git clone https://github.com/kuddukan42/losslessbob
cd losslessbob
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
.venv/bin/python3 run_backend.py
cd gui_next && npm install && npm run dev

Flask backend + Electron/React frontend. This is also the macOS path.

The wizard installs the monthly master database snapshot and (optionally) cached LB site pages for offline browsing. After that, the in-app monthly update keeps checksums current. Optional: SoX for spectrograms, qBittorrent credentials in Setup for one-click torrent adding.
