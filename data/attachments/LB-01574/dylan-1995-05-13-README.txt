BOB DYLAN
Hard Rock Hotel, Las Vegas, NV, May 13, 1995 

This post consists of this readme file, an info file,
14 SHN files, 4 MD5 files (2 for SHNs, 2 for decoded WAVs), 
2 CUE sheets, 2 LOG files, a GIF file, a JPG file,
and 9 PAR2 files.  

You do not need to download the PAR files unless
some of the files are missing or incomplete.
(The base file, dylan-1995-05-13.PAR2, can tell you 
whether any parts are missing.  It is another way to
verify the integrity of your download.)

In case you haven't run into them, see below for 
information about PAR files and recovering missing 
parts of a post.

*****************************************************************************
For more information about the show (e.g., tracklist),  see the INFO.txt file.
*****************************************************************************
All SHN files you have downloaded from this post must be in a 
single directory (folder) when you attempt repair with PAR files
*****************************************************************************

PAR files can be used to recover missing parts of a post - 
it's kind of miraculous!  My experience is that it can be
a bit slow on an older machine, but on a new Pentium 4
the process is pretty quick.

First you need to download a program to handle PAR2 files:
I use Quickpar 0.7.5 from 
  http://www.pbclements.co.uk/QuickPar/
This is a Windows program.  For more options, including
Mac & Linux/Un*x, see
  http://parchive.sourceforge.net/#clients

Note that there are 1.0 & 2.0 versions of PAR.  The
files I'm posting are version 2.0; I'm not sure if the 
non-Windows programs will work.  
(For Macs, you might try
  http://www.xs4all.nl/~gp/MacPAR_deLuxe/ )

For tutorials on the use of PAR2, see
  http://www.pbclements.co.uk/QuickPar/VerifyingAndRepairing.htm
