FIXED VERSION

This disc contains 5 shows from the Dylan/Band 1974 tour -
the "1974 PA Tape Collection" upgrade.  This disc contains
versions that have been fixed by shntool for SBEs - see below.
(I received these discs as SHN files.)

Each top folder contains two folders of SHN files that
expand to two audio discs and an "Extras" folder
that contains artwork, an HTML tracklist, a .CSS file
for the HTML file, CUE files for writing audio CDRs with
CD-text (using EAC, e.g.), and a README file with a bit
of information.

Three of the discs I received had sector boundary errors.
These SBEs were detected and then fixed with shntool.
These are the discs that were fixed:
 MSG 31 Jan 74 evening, disc 1
 Inglewood 14 Feb 74 afternoon, disc 1
 Inglewood 14 Feb 74 evening, disc 1
