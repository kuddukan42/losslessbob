Rollin' Through Stormy Weather - Disc 2 of 2
Bob Dylan
Dandelion DL 125/26
Stadthalle, Vienna, April 30, 1999, plus filler from April & May 1999

Source: original glass-mastered silver CD

Tracks were extracted from a CDR as a single WAV using EAC, then split,
 then run through mkwACT to create SHN files

The CUE files are the file generated by the EAC extraction (Image.CUE)
 and the file corresponding to the EAC splitting into individual WAV files
The latter Cue file can be used for burning decoded WAVs with EAC

The SHN.MD5 file is to verify the SHN files

The SHN files have seek tables (SKT files) appended

The HTML file is a tracklist
