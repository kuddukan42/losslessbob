Pat Garrett & Billy The Kid Sessions - Disc 1 of 2
Bob Dylan
Yassa Remaster
Studio recordings for "Pat Garrett & Billy The Kid" in
 Mexico City, January 20, 1973 and Burbank, February 1973, and
 possibly something from Sam Peckinpah's house in Durango, Mexico

Source/lineage: Yassa remaster, downloaded as FLAC files from
  remasterer as part of vine

***********************************
The only files on this disc that you really need are the 
FLAC files, but there are a few other files that might be 
of interest to some
***********************************

The Cue sheet (CUE file) can be used to write an audio CDR with EAC

The ffp.TXT file is a FLAC fingerprint

The BAT files can be used (under Windows/DOS) to rename files to
  long or short forms

The NFO file came with the download; please refer to it for more
  information

The HTML file is a tracklist
