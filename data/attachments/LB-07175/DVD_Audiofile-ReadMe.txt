***Mac OS X users***

These files are used to create a DVD-A


Use DVD-Audiofile 0.74 by Scott Brown:
http://www.versiontracker.com/dyn/moreinfo/macosx/28674

How to use:
drop flac files or wav files onto the window 
and create a dvd-audio compliant .iso image

notes: will work on all flac files.  
will work on wav files with standard header.  
wav files with non-standard headers won't work

requirements:
java installed, 
twice the hard disk space of the final dvd size 

Click on DVD-Audiofile icon 
Click on Add Files
When files are added you can see if they are 24bit/48kHz
Click on "create .iso"
The .iso file will be created on the desktop
Burn .iso file to DVD with Disk Burner or another burning software.

This will play in DVD-A compliant players.
