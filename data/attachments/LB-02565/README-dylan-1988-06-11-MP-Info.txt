Mountain View, June 11, 1988 - 2 Discs
Bob Dylan
Shoreline Amphitheatre, Mountain View, CA, June 11, 1988

Source/lineage information: taper markp, transfer from the master cassettes
 Nakamichi CM300(cp4) > Sony tcd5m > nakamichi cr-1a > hhb cdr830 >
 EAC > Flac 1.1.2 with tags > easytree (April 2005)

I obtained the audio discs directly from the taper.  I have two other
versions of this show, and they are both different from this one.
The taper indicates that this version has only circulated lightly.
(The Wanted Man boot "Driftin' Too Far From The Shore" doesn't have
the complete concert, and it's not this tape.)

If you want to check this against a copy you have, check the audio
fingerprints toward the end of tracks 2 & 3 on disc 1.
See the annotated tracklist below.

I've included an MD5 file and a Flac fingerprint for the Flac files,
and a log of my EAC extraction (separate files for each disc).

The Extras folder contains a number of items, including MD5 files for
the decoded WAVs, shntool fingerprints for each disc (as a single
WAV file), cue sheets for burning decoded WAV files as an audio CDR,
cue sheets for each single disc WAV file, a "shntool len" output showing
no SBEs or other problems, and a pretty HTML tracklist.

I couldn't find any art.
