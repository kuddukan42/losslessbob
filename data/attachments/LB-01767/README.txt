This DVD contains an assortment of 1999 Dylan shows
in FLAC format.  I have no source/lineage information
for these shows.  In general the shows are as I
received them; there is no re-naming or any of the
other changes I sometimes make (except for Ischgl).

Most of them did not come with an md5 file, but I
created one for every show for quicker verification.
Just be aware these are not original md5 files.

Each folder also contains a shntool information
file.  This was only created to verify that there
were no SBEs or other issues; there is no need to
preserve it.

tracklists on the CDs.  For Dylan/Simon dates,
only Dylan setlists are given; again, I haven't
